Make Some Images

Process:

  1. Make an origional vector image in Corel
  2. Prepare a raster image to be made into a multi-color sticker:
    • Import the raster image into Corel
    • Use the 'outline trace' wizard to convert it to a vector file
    • combine all pieces of the same color
    • Copy and paste each color into its own file
